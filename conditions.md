Basic syntax
```javascript
if (condition) {
    //do something
}
```

else
```javascript
if (condition) {
    //do
} else {
    //do
}
```

else if
``` javascript
if (condition) {
	//do
} else if (other condition) {
	//do
} else {
	//do
}
```

## conditions operators
``` javascript
>=
<=
== // js try to convert both values to the same data type ==> "2" == 2 is true
===
!==
```
