Basic function
```javascript
function sum(x, y) {
    return x + y;
}
```

### Returning booleans
Standard shape
```javascript
function isPassing(grade) {
    if (grade >= 10) {
        return true;
    } else {
        return false;
    }
}
```

Shorter shape
```javascript
function isPassing(grade) {
    return grade >= 10;
}
```
